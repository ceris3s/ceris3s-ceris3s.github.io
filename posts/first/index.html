Hugo 使用

作者
 2023-11-06  2023-11-06  约 123 字  预计阅读 1 分钟 

站点目录下，新建文章，执行：

hugo new xxx.md

用 hugo server 在本地预览

修改完成，确定要上传到GitHub上后，站点目录下执行：

hugo

进行编译，没错误的话修改的内容就顺利同步到public下了，然后**cd public**下，执行提交命令：

git add .
git commit -m ""
git push -u origin main

参考：

https://hugodoit.pages.dev/zh-cn/theme-documentation-basics/
